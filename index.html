what we do

Geotagging

Allows the user to tag their exact geographic location and label routes making the climbs easy to locate.

Photographs

Use your geographical location when taking pictures to pin them to the map and make it easier for people to find the climbs.

Search

Search for climbs in the database and get results for location, routes, pictures and user comments regarding the difficulty of the climb.

IOS/Android

Download on IOS/Android and take the park with you in your pocket. Never get lost again and access all of the functionality of ClimbOn wherever you go.

Functionality

Built with Meteor and React

Complete platform and framework for comprehensive web and mobile applications with javaScript with a user friendly dynamic interface with real-time data updates.

Based on experience

The members of this group enjoy climbing and have run into some pitfalls using other applications. With this knowledge we were able to conceptualize something that would be more user friendly and practical so you don't waste your time out in the park.

Room for improvements

While we fine tune our app, the ideas and goals continue to evolve. Check back as we grow and release onto mobile platforms!
